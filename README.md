
It is a simple expense tracking app to manage your finances. The app should allow users to add, delete, and view their expenses... This solution is based on the page [roadmap.sh](https://roadmap.sh/projects/expense-tracker)

# How to use :
```python
python main.py add --description "<str>" --amount <int>
python main.py list
python main.py summary --month <int> 
python main.py delete --id <int>

```
> [!IMPORTANT]
> Make sure you write the commands correctly so that they do not give errors
